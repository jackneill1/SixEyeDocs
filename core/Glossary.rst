Glossary
********

.. glossary::
  :sorted:

  Device
    A physical item that is SixEye Connected. This means that the manufacturer of that device has allowed the device to talk to the SixEye servers and send information and/or allow interactions from the device to SixEye.

  Device Key
    A string that is used by the device to do the initial authentication with SixEye. Once the key has been used once it cannot be used again and its no longer valid. Keys are valid for 7 days from creation.

  Site
    A collection of information, typically containing some devices, users and the assocaited permissions for those users to view and interact with the devices in that Site. There are typically multiple Sites in a :term:`portal`.

  Portal
    A branded web page where Sites are contained.
